# Недельный meal plan

План пересобран заново после двух проблем в предыдущей версии:

- одинаковый гарнир `рис` в `обед + ужин` в один и тот же день убран полностью
- прошлый креветочный обед с рисом выкинут; вместо него стоит новый batch [Креветки с картофелем и овощами](../../recipes/lunches/krevetki-s-kartoshkoy-i-ovoshchami/README.md)

Дополнительно зафиксировано:

- завтрак понедельника оставлен по факту как [Тосты с ветчиной из индейки](../../recipes/breakfasts/tosty-s-vetchinoy-iz-indeyki/README.md)
- после понедельника индейка из плана убрана
- бананы собраны плотным блоком `чт-сб` на `4` использования: две [Овсянки с бананом](../../recipes/breakfasts/ovsyanka-s-bananom/README.md) и два перекуса [Банан с арахисовой пастой и йогуртом](../../recipes/snacks/banan-arahisovaya-pasta/README.md)
- [Курица с рисом и брюссельской капустой](../../recipes/dinners/kuritsa-ris-bryusselskaya-kapusta/README.md) возвращена на ужины `вт-чт`

Средний день по неделе: примерно `2200 ккал`.
Средние БЖУ по неделе: примерно `Б 172 г / Ж 66 г / У 224 г`.

## План закупок

Закупки ниже уже учитывают:

- завтрак понедельника с индейкой уже был по факту, поэтому в покупки он не входит
- обед понедельника — carry-over [Курицы с рисом и овощами](../../recipes/lunches/kuritsa-ris-ovoshchi/README.md), поэтому ингредиенты на него тоже не входят
- новый креветочный обед идёт без риса, поэтому больше нет дней с одинаковым гарниром в `обед + ужин`
- курица на [Курицу с рисом и брюссельской капустой](../../recipes/dinners/kuritsa-ris-bryusselskaya-kapusta/README.md), [Курицу с рисом и овощами](../../recipes/lunches/kuritsa-ris-ovoshchi/README.md) и [Тёплый салат с курицей и картофелем](../../recipes/dinners/teplyy-salat-s-kuritsey/README.md) закрывается запасом из морозилки, поэтому около `1.5 кг` куриного филе в покупки не добавлено
