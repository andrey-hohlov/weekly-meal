Текущая версия по обедам собрана так: `пн-вт` остатки [Пасты болоньезе](../../recipes/lunches/pasta-boloneze/README.md), `ср-чт` [Картошка с куриными бёдрами](../../recipes/lunches/kartoshka-kurinye-bedra/README.md), `пт-вс` [Курица с рисом и овощами](../../recipes/lunches/kuritsa-ris-ovoshchi/README.md).
