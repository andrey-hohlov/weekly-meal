## План готовок

- Понедельник: приготовить Курица с рисом и овощами (`lunch/kuritsa_ris_ovoshchi`) на 3 порции (обеды пн/вт/ср).
- Понедельник-вечер / вторник-вечер / среда-вечер: приготовить свежий Омлет с лососем (`dinner/omlet_losos`) по 1 порции.
- Четверг: приготовить Паста болоньезе (`lunch/pasta_boloneze`) на 3 порции (обеды чт/пт/сб).
- Четверг: приготовить Курица с овощами на сковороде (`dinner/kuritsa_ovoshchi_skovoroda`) на 2 порции (ужины чт/пт).
- Суббота и воскресенье: приготовить Лосось с овощами (`dinner/losos_ovoshchi`) по 1 порции.
- Воскресенье: приготовить Гречка с индейкой (`lunch/grechka_indeyka`) на 3 порции (обед вс + 2 порции на начало следующей недели).
- Перекусы в основном без готовки; можно заранее разложить порции йогурта, гранолы и орехов.

## Примечания

- В каждый день добавлены два перекуса, чтобы стабилизировать аппетит и не проваливаться по калориям.
- Целевой диапазон по калориям выдержан примерно на уровне 2100–2120 ккал/день.
- Овощной дефолт: в обед ориентир ~200–300 г овощей, в ужин ~150–250 г.
- На тренировочные дни при необходимости можно увеличить порцию одного из перекусов (например, фрукт + орехи).
