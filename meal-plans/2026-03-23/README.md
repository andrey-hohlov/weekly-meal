# Недельный meal plan

План пересобран с понедельника после обеда.
Завтрак и обед понедельника сохранены как уже состоявшийся факт.
Обед вторника тоже зафиксирован: это оставшаяся с выходных [Картошка с куриными бёдрами](../../recipes/lunches/kartoshka-kurinye-bedra/README.md) (`lunch/kartoshka_kurinye_bedra`).

Средний день по неделе: примерно 2190 ккал.

## Примечания

- Готовые йогуртовые перекусы закрывают вторник, среду и пятницу, поэтому отдельно докупать греческий йогурт и гранолу для них не нужно.
- Текущий банан уходит во вторничную овсянку; для субботней нужен ещё один ближе к выходным.
- Лосось из недели по-прежнему убран полностью: не нужно покупать упаковку и потом пытаться растянуть её до выходных.
- Ветчина из индейки из будущих дней убрана полностью: её больше нет ни в меню, ни в закупках.
- Одна порция [Пасты с индейкой и томатным соусом](../../recipes/lunches/pasta-s-indeykoy-i-tomatnym-sousom/README.md) (`lunch/pasta_s_indeykoy_i_tomatnym_sousom`) теперь осознанно переносится на обед понедельника следующей недели, а не уезжает в ужин.
