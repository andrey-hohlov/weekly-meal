# Недельный meal plan

План пересобран с понедельника после обеда.
Завтрак и обед понедельника сохранены как уже состоявшийся факт.
Обед вторника тоже зафиксирован: это оставшаяся с выходных [Картошка с куриными бёдрами](../../recipes/lunches/kartoshka-kurinye-bedra/README.md) (`lunch/kartoshka_kurinye_bedra`).

Средний день по неделе: примерно 2190 ккал (Б 162 г / Ж 67 г / У 216 г).

## Примечания

- Два текущих банана уходят сразу: в понедельничный перекус и во вторничный завтрак.
- Лосось из недели убран полностью: не нужно покупать упаковку и потом пытаться растянуть её до выходных.
- Три готовых йогуртовых перекуса из остатков закрывают среду, четверг и пятницу.
- Горячие блюда собраны блоками: ужины пн–ср, обеды ср–пт, потом ужин чт + обеды сб/вс и форель на выходные.
- Внутри одного дня больше нет повторов типа `лосось + лосось` или `творог + творог`.
