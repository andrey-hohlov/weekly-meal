# Недельный meal plan

План пересобран с понедельника после обеда.
Завтрак и обед понедельника сохранены как уже состоявшийся факт.
Обед вторника тоже зафиксирован: это оставшаяся с выходных [Картошка с куриными бёдрами](../../recipes/lunches/kartoshka-kurinye-bedra/README.md) (`lunch/kartoshka_kurinye_bedra`).

Средний день по неделе: примерно 2160 ккал.

## Примечания

- Три готовых йогуртовых перекуса теперь закрывают вторник, среду и субботу; отдельно докупать гранолу больше не нужно, а дополнительный греческий йогурт нужен только под два банановых перекуса и воскресный перекус с яблоком.
- Творог из оставшейся части недели убран полностью: молочные перекусы закрыты греческим йогуртом и кефиром без отдельной покупки творога.
- С учётом одного уже имеющегося банана нужно докупить ещё четыре: на три овсянки и два банановых перекуса.
- Авокадо из оставшейся части недели убрано полностью: не нужно покупать отдельные плоды ради трёх перекусов.
- Лосось из недели по-прежнему убран полностью: не нужно покупать упаковку и потом пытаться растянуть её до выходных.
- Ветчина из индейки возвращена только на один воскресный завтрак; это не размазывает упаковку по всей неделе и не конфликтует с текущими горячими блюдами.
- На выходные форель убрана полностью: вместо неё один быстрый batch-ужин с курицей, рисом, перцем и кабачком закрывает сразу оба вечера.
- Одна порция [Пасты с индейкой и томатным соусом](../../recipes/lunches/pasta-s-indeykoy-i-tomatnym-sousom/README.md) (`lunch/pasta_s_indeykoy_i_tomatnym_sousom`) теперь осознанно переносится на обед понедельника следующей недели, а не уезжает в ужин.
