# Шаблон рецепта: тосты с лососем

Этот файл является шаблоном. Не использовать как реальный рецепт в планах.

## О рецепте

- Приём пищи: завтрак
- Порций: 1
- Время приготовления: 7 мин

## Пищевая ценность на порцию

- Калории: 520 ккал
- Белки: 28 г
- Жиры: 24 г
- Углеводы: 40 г

## Ингредиенты

- Хлеб цельнозерновой — 80 г
- Слабосолёный лосось — 70 г
- Творожный сыр безлактозный — 30 г
- Авокадо — 70 г
- Помидор — 100 г

## Как готовить

1. Подсушить хлеб.
2. Намазать творожный сыр.
3. Выложить лосось и авокадо.
4. Подать с нарезанным помидором.
