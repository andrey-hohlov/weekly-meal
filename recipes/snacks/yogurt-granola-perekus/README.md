# Йогурт с гранолой (перекус)

## О рецепте

- Приём пищи: перекус
- Порций: 1
- Время приготовления: 2 мин

## Пищевая ценность на порцию

- Калории: 260 ккал
- Белки: 16 г
- Жиры: 7 г
- Углеводы: 32 г

## Ингредиенты

- Йогурт греческий безлактозный — 150 г
- Гранола — 25 г
- Ягоды — 50 г

## Как готовить

1. Выложить йогурт в миску.
2. Добавить гранолу и ягоды.
3. Есть сразу, чтобы гранола осталась хрустящей.
