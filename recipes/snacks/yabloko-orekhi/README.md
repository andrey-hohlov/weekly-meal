# Яблоко и орехи

## О рецепте

- Приём пищи: перекус
- Порций: 1
- Время приготовления: 3 мин

## Пищевая ценность на порцию

- Калории: 220 ккал
- Белки: 4 г
- Жиры: 13 г
- Углеводы: 24 г

## Ингредиенты

- Яблоко — 180 г
- Орехи — 20 г

## Как готовить

1. Нарезать яблоко дольками.
2. Подать с орехами.
