# Курица с овощами на сковороде

Быстрый и лёгкий ужин на 2 порции.

## Ингредиенты

- Куриное филе — 400 г
- Болгарский перец — 1 шт
- Кабачок — 1 небольшой
- Соевый соус — 1 ст.л.
- Растительное масло — 1 ч.л.

## Приготовление

1. Нарезать курицу кубиками.
2. Нарезать овощи.
3. Разогреть масло на сковороде.
4. Обжарить курицу до готовности.
5. Добавить овощи и готовить ещё 5–7 минут.
6. Добавить соевый соус и перемешать.
