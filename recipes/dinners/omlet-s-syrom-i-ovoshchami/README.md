# Омлет с сыром и овощами

Быстрый ужин, готовится за несколько минут.

## Ингредиенты

- Яйца — 3 шт
- Сыр — 40 г
- Помидор — ~1 средний
- Шпинат или зелень — немного
- Оливковое масло — 1 ч.л.

## Приготовление

1. Разогреть сковороду с небольшим количеством масла.
2. Нарезать помидор.
3. Взбить яйца.
4. Вылить яйца на сковороду, добавить помидор и зелень.
5. Посыпать тёртым сыром.
6. Готовить до готовности.

Хорошо сочетается с овощным салатом.
