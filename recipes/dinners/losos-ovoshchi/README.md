# Лосось с овощами

Полезный ужин с рыбой.

## Ингредиенты

- Филе лосося — ~180 г
- Брокколи — 150 г
- Кабачок — 1 небольшой
- Лимон — немного сока
- Оливковое масло — 1 ч.л.

## Приготовление

1. Посолить и поперчить лосось.
2. Обжарить рыбу на сковороде или запечь ~10–12 минут.
3. Овощи можно слегка обжарить или приготовить на пару.
4. Подать с лимонным соком.
