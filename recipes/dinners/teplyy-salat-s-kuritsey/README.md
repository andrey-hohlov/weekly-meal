# Тёплый салат с курицей

Лёгкий ужин с большим количеством овощей.

## Ингредиенты

- Куриное филе — 200 г
- Листья салата
- Огурец
- Помидор
- Оливковое масло

## Приготовление

1. Обжарить куриное филе на сковороде до готовности.
2. Нарезать овощи.
3. Выложить овощи на тарелку.
4. Сверху добавить горячую курицу.
5. Полить оливковым маслом.
