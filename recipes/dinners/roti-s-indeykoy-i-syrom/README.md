# Роти с индейкой и сыром

Плотный и быстрый ужин из готовых продуктов. Хороший вариант, когда хочется чего-то поинтереснее обычной базы, но лучше оставлять его как редкий ужин, а не повседневную основу.

## Ингредиенты

- Лепёшка роти — 1 шт
- Ветчина из индейки — 75 г
- Сыр — 40 г
- Творожный сыр — 30 г
- Помидор — немного
- Зелень — немного

## Приготовление

1. Обжарить лепёшку роти с одной стороны
2. Перевернуть, выложить ветчину из индейки, сыр, помидор и зелень
3. Довести до готовности, свернуть
4. Выложить на тарелку, дать остыть пару минуть, затем намазать творожный сыр.

## Заметки

Если хочется сделать вариант полегче, можно немного уменьшить обычный сыр или оставить только один вид сыра.
