# Паста болоньезе

## О рецепте

- Приём пищи: обед
- Порций: 3
- Время приготовления: 25 мин

## Пищевая ценность на порцию

- Калории: 640 ккал
- Белки: 45 г
- Жиры: 18 г
- Углеводы: 68 г

## Ингредиенты

- Фарш говяжий — 500 г
- Паста — 270 г
- Томатный соус — 350 г
- Лук — 120 г
- Оливковое масло — 10 г

## Как готовить

1. Отварить пасту.
2. Обжарить лук на масле.
3. Добавить фарш и готовить до готовности.
4. Влить томатный соус, прогреть несколько минут и подать с пастой.
