# Паста болоньезе

## О рецепте

- Приём пищи: обед
- Порций: 3
- Время приготовления: 25 мин

## Пищевая ценность на порцию

- Калории: 576 ккал
- Белки: 41 г
- Жиры: 16 г
- Углеводы: 61 г

## Ингредиенты

- Фарш говяжий — 450 г
- Паста — 243 г
- Томатный соус — 315 г
- Лук — 108 г
- Оливковое масло — 9 г

## Как готовить

1. Отварить пасту.
2. Обжарить лук на масле.
3. Добавить фарш и готовить до готовности.
4. Влить томатный соус, прогреть несколько минут и подать с пастой.
