# Гречка с индейкой

## О рецепте

- Приём пищи: обед
- Порций: 3
- Время приготовления: 25 мин

## Пищевая ценность на порцию

- Калории: 539 ккал
- Белки: 38 г
- Жиры: 20 г
- Углеводы: 54 г

## Ингредиенты

- Гречка — 200 г
- Фарш индейки — 450 г
- Морковь — 100 г
- Лук — 100 г
- Растительное масло — 8 г

## Как готовить

1. Отварить гречку до готовности.
2. На сковороде с маслом обжарить лук и морковь.
3. Добавить фарш индейки и готовить до полной готовности.
4. Смешать с гречкой и разложить по порциям.
