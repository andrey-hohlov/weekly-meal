# Гречка с индейкой

Простой и сытный обед.

## Ингредиенты (на 3 порции)

- Гречка — 240 г
- Фарш индейки — 500 г
- Морковь — 120 г
- Лук — 120 г
- Растительное масло — 10 г

## Приготовление

1. Отварить гречку.
2. На сковороде обжарить лук и морковь.
3. Добавить фарш индейки и готовить до готовности.
4. Смешать с гречкой.

## Заметки

Хорошо подходит для batch cooking.
