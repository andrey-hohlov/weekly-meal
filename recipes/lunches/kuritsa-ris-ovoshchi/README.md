# Курица с рисом и овощами

Простой обед, который удобно готовить сразу на 2–3 дня.

## Ингредиенты (на 3 порции)

- Куриное филе — 600 г
- Рис — 240 г
- Морковь — 120 г
- Лук — 120 г
- Соевый соус — 20 мл
- Растительное масло — 10 г

## Приготовление

1. Отварить рис.
2. Нарезать куриное филе кубиками.
3. На сковороде разогреть масло, обжарить лук и морковь.
4. Добавить курицу и готовить до готовности.
5. Добавить соевый соус и перемешать.
6. Подавать с рисом.

## Заметки

Хорошо разогревается на следующий день.
