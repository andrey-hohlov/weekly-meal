# Лосось с рисом

Простой рыбный обед.

## Ингредиенты (на 2 порции)

- Филе лосося — 300 г
- Рис — 160 г
- Лимон — немного сока
- Оливковое масло — 10 г

## Приготовление

1. Отварить рис.
2. Лосось посолить, поперчить, сбрызнуть лимоном.
3. Обжарить или запечь ~10–12 минут.
4. Подавать с рисом.

## Заметки

Полезно иногда добавлять рыбные блюда в рацион.
