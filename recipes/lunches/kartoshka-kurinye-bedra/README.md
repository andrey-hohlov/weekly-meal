# Картошка с куриными бёдрами

Очень простое домашнее блюдо.

## Ингредиенты (на 3 порции)

- Куриные бёдра — 600 г
- Картофель — 800 г
- Лук — 120 г
- Оливковое масло — 10 г

## Приготовление

1. Нарезать картофель крупными кусками.
2. Нарезать лук.
3. Выложить всё на противень.
4. Добавить куриные бёдра.
5. Полить оливковым маслом, посолить и поперчить.
6. Запекать ~35–40 минут при 200°C.

## Заметки

Можно готовить сразу на несколько дней.
