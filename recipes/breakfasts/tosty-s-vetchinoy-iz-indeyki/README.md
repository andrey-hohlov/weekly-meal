# Тосты с ветчиной из индейки

## О рецепте

- Приём пищи: завтрак
- Порций: 1
- Время приготовления: 7 мин

## Пищевая ценность на порцию

- Калории: 410 ккал
- Белки: 27 г
- Жиры: 12 г
- Углеводы: 42 г

## Ингредиенты

- Хлеб цельнозерновой — 80 г
- Ветчина из индейки — 75 г
- Сыр творожный — 30 г
- Помидор — 100 г
- Огурец — 80 г

## Как готовить

1. Подсушить хлеб.
2. Намазать творожный сыр.
3. Сверху выложить ветчину из индейки и овощи.
4. Подать сразу.
