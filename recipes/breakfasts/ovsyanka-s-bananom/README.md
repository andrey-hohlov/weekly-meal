# Овсянка с бананом

## О рецепте

- Приём пищи: завтрак
- Порций: 1
- Время приготовления: 5 мин

## Пищевая ценность на порцию

- Калории: 480 ккал
- Белки: 18 г
- Жиры: 16 г
- Углеводы: 65 г

## Ингредиенты

- Овсяные хлопья — 60 г
- Молоко безлактозное — 200 мл
- Банан — 100 г
- Арахисовая паста — 15 г

## Как готовить

1. Сварить овсяные хлопья на безлактозном молоке до нужной консистенции.
2. Добавить нарезанный банан.
3. Сверху выложить арахисовую пасту.
